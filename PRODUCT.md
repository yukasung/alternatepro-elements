# Product

## Register

product

## Users

WordPress site builders and administrators who use Elementor Free and want richer theme-building and widget capabilities without depending on Elementor Pro internals.

## Product Purpose

AlternatePro Elements extends Elementor Free with plugin-owned widgets, header/footer template rendering, theme-builder foundations, conditions, dynamic data, and admin settings. Success means site builders can create predictable WordPress layouts with Elementor Free public APIs, safe defaults, and maintainable plugin code.

## Brand Personality

Practical, compatible, dependable. The product should feel familiar to Elementor users while staying original, transparent, and production-focused.

## Anti-references

Do not copy, inspect, reuse, or derive from Elementor Pro source code. Avoid overbuilt controls, visual gimmicks, brittle private API dependencies, and UI behavior that surprises experienced WordPress builders.

## Design Principles

- Preserve Elementor familiarity without cloning protected implementation.
- Favor simple, stable controls that map clearly to frontend output.
- Make defaults usable before customization.
- Keep behavior accessible, responsive, and respectful of reduced motion.
- Prefer maintainable WordPress and Elementor Free public APIs over clever shortcuts.

## Accessibility & Inclusion

Use semantic WordPress output where possible, maintain keyboard-usable controls, preserve ARIA state for interactive widgets, and support reduced-motion preferences for animation.
