# AlternatePro Elements - Codex Commands

## Quick Workflow

Use these short commands for normal development.

### Start / Implement

```text
Read docs/context.md

Verify current phase dashboard.

Implement current task.

Follow architecture.md.
Follow shared abstraction rules.
Avoid duplicate logic.

After implementation, update:
- docs/status.md
- CHANGELOG.md
- relevant docs/releases/* if applicable
- relevant docs/dashboards/* if applicable
```

### Review

```text
Read docs/agents/review.md
```

### Test

```text
Read docs/agents/test.md
```

### Security Review

```text
Read docs/agents/security.md
```

### Refactor If Needed

```text
Read docs/agents/refactor.md
```

### Complete Task

```text
Read docs/done.md

Verify:
- docs/status.md updated
- CHANGELOG.md updated
- release documentation updated if applicable
- dashboards updated
- next task identified
```

### Check Status

```text
Read docs/status.md

Summarize:
- Current Phase
- Completed Work
- Remaining Work
- Open Issues
- Recommended Next Task
```
